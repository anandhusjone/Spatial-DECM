Instructions

Layer Tools

Set a layer into edit mode from the Layers panel to start editing.

Use the + button in Layers to browse files or create a new named layer, and the + button in the attribute table to add a field.

Draw & Edit

Use the draw controls on the map to add points, lines, or polygons into the editable layer. Export is available from each layer in the Layers panel.

Tips

  • Upload several files to compare them side by side.
  • Set one layer as editable before drawing or moving vertices.
  • Use the edit tool to move nodes on lines and polygons.
  • Edit attributes directly in the table for the active edit layer.
  • CSV files should include latitude and longitude columns.
